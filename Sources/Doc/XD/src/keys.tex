\section{Window management}

\KeyListBegin{8 cm}

\KeyDef{Ctrl+T} {Activate Program window}
\KeyDef{Ctrl+R} {Activate Registers window}
\KeyDef{Ctrl+B} {Activate Breaks window}
\KeyDef{Ctrl+M} {Activale Module List window}
\KeyDef{Ctrl+L} {Activate Procedure List window}
\KeyDef{Ctrl+V} {Activate Global variables window}
\KeyDef{Ctrl+D} {Open a new Memory Dump window}
\KeyDef{Ctrl+W} {Enter window move/resize mode (see below)}
\KeyListEnd{}

In the window move/resize mode, arrow keys move window,
{\bf Ctrl}+arrow keys change its size. {\bf Enter} or
{\bf Ctrl+W} exits the mode.
